### Claim GBGL

1. Sign in via Metamask and create new proposal as shown in the [tutorial](https://github.com/alekcangp/BGL-DAO/blob/main/tutorial.gif)

2. Wait until the voiting period ends (about for a day)

3. Execute proposal.
